    Creating a logical backup:

    mariadb-dump -h <host> -u <user> -p \
--events \
--routines \
--triggers \
--add-drop-database \
--compress \
--hex-blob \
--opt \
--skip-comments \
--single-transaction \
--databases <database_name> > backup.sql

    Restoring a logical backup: 

    mariadb <host> -u <user> -p < backup.sql
